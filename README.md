## 🌟 Overview

EquipSync is a hyperlocal peer-to-peer equipment rental marketplace designed to transform idle assets into productive resources.

The platform enables users to both rent equipment and list their own tools for rent, creating a decentralized and community-driven sharing ecosystem.

From engineering project kits and electronics to photography gear and event equipment, EquipSync provides flexible booking with intelligent scheduling and dynamic delivery pricing.

---

## 🎯 Problem Statement

Across India:

- Engineering students cannot afford expensive project tools.
- Freelancers and startups struggle with high capital investment.
- Event organizers require short-term access to costly equipment.
- Many professional tools remain idle and underutilized.
- Existing rental platforms lack hyperlocal intelligence and peer-to-peer accessibility.

There is a need for a unified intelligent system that connects equipment owners with renters efficiently and affordably.

---

## 💡 Our Solution

EquipSync provides a smart digital marketplace where:

✔ Any registered user can rent equipment  
✔ Any registered user can list their own equipment for rent  
✔ Smart booking logic prevents double booking conflicts  
✔ Delivery pricing adjusts dynamically based on proximity  
✔ Users can act as both renter and equipment owner  

This creates a circular economy model that increases asset utilization and reduces ownership costs.

---

## 🤝 Peer-to-Peer Model

EquipSync operates on a fully community-driven peer-to-peer rental ecosystem.

Any user can:

- List tools, electronics, or machinery for rent
- Browse and rent equipment from nearby users
- Generate income from idle assets
- Access equipment without long-term investment

This decentralized structure ensures scalability and hyperlocal growth.

## 🔥 Key Features

### 🔐 Authentication System
- Secure Sign Up & Sign In
- LocalStorage-based user management
- User session handling

### 🛒 Equipment Marketplace
- Category filtering
- Search functionality
- Equipment detail view
- Dynamic equipment listing

### 📅 Smart Booking System
- Start and End date selection
- Conflict detection (prevents booking unavailable dates)
- Intelligent date validation

### 🚚 Dynamic Delivery Logic
- User enters distance from owner
- Free doorstep delivery within 4KM
- ₹100 delivery charge beyond 4KM
- Live price breakdown display

### 💰 Real-Time Price Calculation
- Rental cost = Price × Number of days
- Delivery charge added dynamically
- Transparent cost breakdown

### ➕ Add Equipment for Rent
- Logged-in users can add new equipment
- Category selection
- Price per day entry
- Description & image upload (UI level)
- Equipment added dynamically to marketplace

### ✅ Booking Confirmation
- Auto-generated Booking ID
- Full booking summary display
- Address & contact details included
- Success confirmation interface
